    :contains - jQuery 홈페이지 :contains API 문서 : http://api.jquery.com/contains-selector/

    :contains 셀렉터는 지정한 문자열이 포함된 요소를 찾습니다.

    • 관련 CSS
    • borderborder-radiusmarginmargin-toppaddingtext-alignwidth
    • jQuery
    • $( '셀렉터:contains(찾을문자)' ) - 해당 셀렉터에 찾을문자가 포함된 것을 찾습니다.
    • $( '.contains_test div:contains(찾을문자)' ) - 클래스명이 contains_test 인 요소의 div 중 찾을문자가 포함된 것을 찾습니다.
    • $( '.contains_test div:contains(강아지)' ) - 클래스명이 contains_test 인 요소의 div 중 문자열 강아지가 포함된 것을 찾습니다.
    • - 태그의 속성값 등이 아닌 포함하고 있는 문자열을 대상으로 합니다.

    :empty 셀렉터는 비어있는 요소를 찾습니다.

    • borderborder-radiusmarginmargin-toppaddingtext-alignwidth
    • jQuery
    • $( '셀렉터:empty' ) - 셀렉터에서 비어있는 요소만 찾기
    • $( '.empty_test div:empty' ) - 클래스명이 empty_test 인 요소의 DIV 중 비어있는것 찾기

    :enabled - jQuery 홈페이지 :enabled API 문서 : http://api.jquery.com/enabled-selector/

    :enabled 셀렉터는 활성화 되어있는 폼요소를 찾습니다.
    즉 disabled 된 요소의 반대입니다.
    disabled 속성이 지정되지 않은 평범한 폼 요소를 생각하면 됩니다.

    • 관련 CSS
    • borderborder-radiusmarginmargin-toppaddingtext-alignwidth
    • jQuery
    • $( '셀렉터:enabled' ) - 셀렉터에 해당하는 요소중 enabled(활성) 된것을 찾습니다.
    • $( '.enabled_test input:enabled, .enabled_test textarea:enabled' ) - 텍스트 박스와 텍스트에리어를 대상으로 enabled 찾기
    • $( '.enabled_test :enabled' ) - 클래스명이 enabled_test 인 요소에서 모든 enabled 찾기
    • $( '.enabled_test :not(:disabled)' ) - 이렇게 not 셀렉터를 사용해도 위와 같은 처리를 합니다. (반대의 반대는 긍정과 같은)

    :eq - jQuery 홈페이지 :eq API 문서 : http://api.jquery.com/eq-selector/

    :eq 셀렉터는 요소를 순서로 접근하여 찾습니다.

    • 관련 CSS
    • margin-top / width
    • jQuery
    • $( '셀렉터:eq(숫자)' ) - 셀렉터에 해당하는 요소중 특정 순서의 것을 찾습니다.
    • $( '셀렉터:eq(0)' ) - 셀렉터에 해당하는 요소중 첫번째것 (0부터 시작)
    • $( '셀렉터:eq(2)' ) - 셀렉터에 해당하는 요소중 세번째것
    • $( '셀렉터:eq(마이너스 숫자)' ) - 셀렉터에 해당하는 요소중 특정 뒤로부터 순서의 것을 찾습니다.
    • $( '셀렉터:eq(-1)' ) - 셀렉터에 해당하는 요소중 마지막것
    • $( '셀렉터:eq(-2)' ) - 셀렉터에 해당하는 요소중 마지막에서 두번째것
    • $( '.eq_test img:eq(0)' ) - 클래스명이 eq_test 인것에 속한 이미지중 첫번째 것 찾기
    • $( '.eq_test img:eq(' + n + ')' ) - 예제에서는 변수를 받아서 문자열 처리해 원하는것을 찾았습니다.
    :even - jQuery 홈페이지 :even API 문서 : http://api.jquery.com/even-selector/

    :even 셀렉터는 짝수번째 요소를 찾습니다.
    순서는 0부터 시작한다는것에 유의하세요. 0포함 짝수 찾기.

    • 관련 CSS
    • margin-top / width
    • jQuery
    • $( '.even_test img:even' ) - 짝수번째 요소를 찾습니다.
    • - 예제에서는 홀수번째 얼굴들이 반응했습니다.
    • - even 셀렉터는 0 포함 짝수번째를 찾습니다. 0, 2, 4 이런 순서를 찾는것이죠.
    • - 맨앞에 것이 반응했어도 0부터 시작이므로 바르게 찾아진 것입니다.
    • - 기억하세요 0 부터 시작이다 !
    • 관련 메소드 : .slideToggle()
    • 관련 예제 : easing(이징) 효과 모음
    :file - jQuery 홈페이지 :file API 문서 : http://api.jquery.com/file-selector/

    :file 셀렉터는 폼 요소중 파일입력폼을 찾습니다.

    • 관련 CSS
    • border / border-radius / margin / margin-top / padding / text-align / width
    • jQuery
    • $( '.file_test :file' ) - 클래스명이 file_test 인 요소에서 파일입력폼만 찾습니다.
    • - 파일 선택폼은 읽기 전용입니다.
    • - 직접 파일을 선택하고 예제의 값 읽기 버튼을 누러보면 해당 파일의 경로나 이름이 보입니다.
    • - 이때 경로가 좀 다르게 보일 수 있습니다. 보안문제 때문입니다.
    • - 예제에서 값 쓰기 버튼을 눌러도 값이 입력되지 않습니다.
    • - 그리고 값을 임의로 넣을 수 없는 이유도 중요한 보안문제 때문입니다.
    • - 만약 값을 넣을 수 있다면 이런게 가능해 집니다. 가령 어떤 페이지에 접속했더니 ...
    • - 윈도우의 특정 파일의 경로를 스크립트로 설정하고 전송까지 해버린다면 어떻게 될까요?
    • - 그런 의미에서 파일 전송폼의 찾기 버튼을 이미지로 바꾸는것도 대단히 어렵습니다.

    :gt - jQuery 홈페이지 :gt API 문서 : http://api.jquery.com/gt-selector/

    :gt 셀렉터는 해당하는 요소중 지정한 순서 보다 큰것을 찾습니다.
    순서는 0 부터 시작합니다. 그리고 지정한 순서는 포함하지 않습니다.
    gt 는 Greater Than, 뭐뭐 보다 큰것 입니다.

    • 관련 CSS
    • border / border-radius / margin-top / padding / position / width
    • jQuery
    • $( '셀렉터:gt(숫자)' ) - 기본 사용예
    • $( '.gt_test img:gt(0)' ) - 클래스명이 gt_test 인 요소안의 이미지중 순서가 0 보다 큰것을 찾습니다.
    • - 순서는 0 부터 시작합니다. 0 보다 큰것은 순서 1에 해당하므로 두번째 것을 찾습니다.
    • $( '.gt_test img:gt(2'):not(:animated)' )
    • - 클래스명이 gt_test 인 요소안의 이미지중 순서가 2번째 보다 큰것중 현재 애니메이션 되지 않는것을 찾아라!
    • - 이렇게 하면 버튼을 연타하더라도 애니메이션 중인 대상은 포함하지 않으므로 오작동을 방지 할 수 있습니다.
    • 예제 관련 메소드 : .animate()
